void EuropeanVanillaOption::calc_d_ ( )
inlineprivate

Calculate intermediate variables:

\( \displaystyle d_1 = \frac{ln(S/K) + (r + \sigma^2 / 2)T}{\sigma \sqrt{T}} \), represents a standardized measure of how far the current price S is from the strike price K after accounting for the time to maturity, risk-free rate, and volatility.

\( \displaystyle d_2 = d_1 - \sigma \sqrt{T} \), reflects the uncertainty (volatility) over the time to maturity.
